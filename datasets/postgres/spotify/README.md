# Spotify Playlist!

## Description

This dataset is a representation of playlists in a Spotify account.  It includes these tables:

- Album
- Artist
- Category
- Genre
- Image
- Playlist
- Track
- User

## How to use

Run:

```sh
psql -U username -d spotify < spotify.pgsql 
```

For more detailed instructions, go [here](../README.md)
